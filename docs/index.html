Flags API Documentation - NpSync

Comprehensive guide to using our Flags API

Introduction

Welcome to our API documentation. Here you'll find information on how to use our flags API in your projects under 1 minute. Ready to use endpoints out of the box.

BASE URL : https://flags.npsync.com/v1
HTTP Method : GET

Authentication

Our Flag API is free and doesn't require authentication. You can start using it right away.

Variants

Our API supports multiple size and type variants

Key Values
SIZE
(px) 		w20, w40, w80, w160, w320, w640, w1280, w2560

h20, h24, h40, h48, h60, h80, h120, h160, h240
EXT png, webp, jpg
CODE ISO 3166 country code

Eg: us, np, in, etc

Endpoints

Images (SVG)

Programatically get the SVG flag images hosted from our CDN.

GET /countries/{CODE}.svg

Examples:

GET /countries/np.svg
GET /countries/us.svg
GET /countries/in.svg

Example response:

https://flags.npsync.com/v1/countries/np.svg



https://flags.npsync.com/v1/countries/us.svg



https://flags.npsync.com/v1/countries/in.svg


Images (PNG, JPG, WEBP)

Programatically get the flag images hosted from our CDN.

GET /countries/{SIZE}/{CODE}.{EXT}

Examples:

GET /countries/w160/np.png
GET /countries/w320/us.webp
GET /countries/h160/in.jpg

Example response:

https://flags.npsync.com/v1/countries/w160/np.png



https://flags.npsync.com/v1/countries/w320/us.webp



https://flags.npsync.com/v1/countries/h160/in.jpg


Zips

Want to self host the flags? Download the zip files with the extension of your choice.

GET /countries/zips/{EXT}/{SIZE}.zip

Examples:

GET /countries/zips/jpg/w640.zip
GET /countries/zips/webp/h48.zip

Example response:

https://flags.npsync.com/v1/countries/zips/png/w640.zip

https://flags.npsync.com/v1/countries/zips/webp/h48.zip
